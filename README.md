# devenv
All development environment setup files go here.
Those includes configs for OS, VM, Docker, App configs.
It includes dotfiles, and may include more, such as install scripts.


How to use:

Step 1:
source cfg-init

Step 2:
echo ".cfg" >> $HOME/.gitignore
